# Starting project with Makefile
```bash
make init
```


# Starting project

```bash
# start containers
docker-compose up -d
# start node server
npm run start:dev
```


# Mongo doc

```bash
# going into container mongo
docker exec -ti mongo bash
# connect to db mongo
mongosh
```
